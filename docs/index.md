## 核心公式速查

| 公式 | 用途 |
|-----|------|
| $A = P \times (1+r)^n$ | 复利终值 |
| $PE = \text{市值} / \text{盈利}$ | 市盈率（越低越便宜） |
| $E/P = \text{盈利} / \text{市值} = 1/PE$ | 盈利收益率（> 10% 买，< 6.4% 卖） |
| $PB = \text{市值} / \text{净资产}$ | 市净率（周期行业专用） |
| $ROE = \text{净利润} / \text{净资产}$ | 净资产收益率（越高越优质） |
| $\text{股息率} = \text{现金分红} / \text{市值}$ | 分红收益率 |
| $\text{分红率} = \text{现金分红} / \text{净利润}$ | 公司派息比例（≠股息率） |
| 年复合收益 ≈ 初始股息率 + ΔPE/年 + 盈利增长率 | 博格公式：指数收益三来源 |

## 买卖决策速查表

### 怎么选（四大估值方法适用指数）

| 策略 | 适用指数 |
|------|---------|
| **盈利收益率法**（E/P > 10% 买）| 上证50、上证红利、中证红利、基本面50、央视50、50AH优选、恒生、H股 |
| **博格公式（PE）** | 沪深300、中证500、创业板指、必需消费、医药、可选消费、养老产业 |
| **博格公式变种（PB）** | 证券行业、银行行业、金融行业、非银金融、地产行业 |
| **直接放弃** | 长期亏损的指数 |

### 怎么买（懒人定投三档操作）

| 盈利收益率 E/P | 操作 |
|--------------|------|
| **> 10%** | ✅ 继续定投（低估区） |
| 6.4% ~ 10% | ⏸ 暂停定投，坚定持有 |
| **< 6.4%** | 🔴 分批卖出（高估区） |

### 怎么卖（分批止盈）
- 触发信号：E/P < 6.4% 或 PE 进入历史高分位（> 80%）
- 操作方式：将总持仓分10等份，每触发一次卖出1份
- 卖出后：等待下一轮低估机会重新入场
