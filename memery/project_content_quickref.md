# 各章核心内容要点速查

**Why:** 方便在后续对话中快速检索某个概念或章节的核心结论，避免重新阅读全书。

---

## 是什么（概念类）

- **基金** = 众人集资，托管银行保管资金，基金经理运作投资
- **NAV** = 基金总资产 ÷ 总份额（每日收盘后公布）
- **指数基金** = 机械复制指数成分股和权重，不做主观选股
- **ETF** = 场内交易的指数基金，实时价格，费率极低
- **PE** = 总市值/年净利润（多少年回本），越低越便宜
- **PB** = 总市值/净资产（账面价格几折），PB<1为极低估

## 怎么选（筛选类）

**指数基金选法**：
1. 查PE/PB历史百分位（集思录/且慢），低于30%为低估区
2. 宽基优先（沪深300/中证500/标普500）
3. 费率最低的同类产品

**主动基金21项筛选**（需满足15项以上）：
- 经理任职≥3年；在管≤3只；最大回撤≤30%
- 5年年化跑赢基准≥3%；夏普比率≥1
- 规模10~100亿；前十大持仓≤60%

## 怎么买（交易类）

- **定投**：月定投为主，发薪后第2天自动扣款
  - PE低估区：投2倍；合理区：1倍；高估区：0.5倍
- **渠道**：天天基金（1折申购费，支持自动定投）
- **ETF**：需证券账户，在东方财富证券/主流券商买
- **建仓**：分3~5批，间隔2~4周，不一次性满仓

## 怎么卖（止盈/止损类）

**止盈策略**：
- 目标收益达到30~50%时，分批赎回（先赎1/3，再1/3，持续观察）
- PE百分位>80%时，逐步减仓

**止损策略**：
- 单只基金亏损>20%，减半仓位
- 主动基金持有>3年且连续落后同类20%以上，考虑更换

**再平衡**：当任一资产偏离目标比例超过5个百分点时执行

## 关键数字速查

| 数字 | 含义 |
|-----|------|
| 72/8 = 9年 | 年化8%约9年翻倍 |
| 1.5%×20年 | 主动基金费率比ETF多损耗约10万元（10万本金）|
| 30% | 偏股混合型基金最大回撤上限（超过不选）|
| 10~100亿 | 主动基金最优规模区间 |
| 15/21项 | 主动基金筛选最低通过分 |
| 5% | 仓位偏离再平衡触发阈值 |
| 3~6个月 | 应急备用金规模（不参与投资）|
| 72小时 | 有操作冲动时的强制等待期 |
